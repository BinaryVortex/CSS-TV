## ✨ Overview

**CSS-TV** is a creative frontend project that recreates a vintage television interface using pure HTML and CSS. The project demonstrates advanced CSS techniques such as custom shapes, gradients, shadows, animations, and responsive layouts.

## 🚀 Features

- 📺 Retro television design
- 🎨 Pure HTML & CSS implementation
- ✨ Smooth animations and transitions
- 📱 Responsive layout
- 🌈 Modern CSS effects
- ⚡ Lightweight with no dependencies

## 🛠️ Built With

- HTML5
- CSS3

## 🚀 Getting Started

```bash
git clone https://github.com/BinaryVortex/CSS-TV.git
cd CSS-TV
```

Open `index.html` in your browser.
